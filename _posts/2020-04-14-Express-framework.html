node.js 자체는 server가 아니지만 node.js의 framework를 통하여 http server를 만들 수 있다.

1. express 설치하기

먼저 프로젝트 폴더를 생성한다.

{% highlight Bash %} $ mkdir nodejs // 프로젝트 폴더 생성 $ cd nodejs $ npm init // npm을 통해 project initialization $ npm i express // express 설치 {% endhighlight %}
